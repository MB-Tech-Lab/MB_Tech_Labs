# Admin Dashboard

## Purpose
The Admin Dashboard is the centralized command center for MB Tech Labs administrators to monitor work activity, performance, and system health.

## Dashboard Sections

### User Overview
- Total active interns and employees
- User status: online, offline, idle
- Attendance compliance and exceptions
- Role assignments and team membership

### Task Tracking
- Global task board with status filters
- Overdue and upcoming deadlines
- High-priority task alerts
- Task submission and approval queue

### Performance Insights
- Aggregate completion rates
- Quality review statistics
- Time utilization and idle time reports
- Low performance and inactivity flags

### Reports & Analytics
- Daily work summaries
- Weekly performance snapshots
- Individual performance reports
- Team and role-based analytics

### System Activity
- Recent activity logs and audit trails
- Submission timestamps and review actions
- Work session start/stop history
- Policy violation alerts

## Admin Actions
- Assign, reassign, and prioritize tasks
- Approve or reject task submissions
- Flag and escalate low performance
- Review time logs and activity details
- Export reports for review and compliance

## Visibility and Control
- Admin has full visibility over all users, tasks, and activity.
- Access is restricted to authorized administrative users.
- Sensitive data is displayed only where relevant to monitoring and decision-making.

## Alerts and Flags
- **Low performance**: Users with poor completion or quality metrics
- **Inactivity**: Users with prolonged gaps between logins or time entries
- **Deadline risk**: Tasks approaching or past due dates
- **Policy issues**: Time entry anomalies or unacceptable use alerts

## Administrative Standards
- Use dashboard data to coach and improve team performance.
- Do not misuse visibility for unrelated surveillance.
- Maintain confidentiality of user performance data.
